Тема проекта: Сайт интернет-магазина
Для сборки проекта необходимы библиотеки, указанные в файле requirements.txt, а также файлы, находящиеся в папках data, templates, static, forms.

На главной странице сайта находятся кнопки для перехода в определенные категории. Также вверху есть поиск, осуществляемый по названиям всех товаров.
На страницах категорий и поиска находятся фильтры, для более точного поиска.
При нажатии на иконку "Penguins Corp" происходит переход на главную страницу.
В личном кабинете можно поменять данные о себе и создать свой магазин (при условии, что пользователю есть 18 лет и он подтвердил свою почту).
На странице магазина можно добавлять товары, удалять и редактировать их.
На странице товара можно оставить отзыв (редактировать и удалять его нельзя), причем только один, и добавить товар в корзину.
Из корзины есть возможность оплатить товары и подождать доставку 10-20 лет.
Внизу страницы находится информация о разработчиках и сайте, форма для отправки вопроса.
